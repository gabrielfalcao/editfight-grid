Welcome to EditFight.com!

This week it's an MMO pixel painting game. Basically everyone sees the same 100 x 100 grid and we can all edit it simultaneously.

I've added a time-lapse feature so it's recording every change to the grid! I plan to make a script that turns each 24 hour period into a gif/video and posts it to imgur/youtube, linking them back here!

Please be respectful and keep it family friendly, I'd like to show my kids the time-lapses, and drawing obscenities is the equivalent of telling a poop joke, and I'm sure you're more creative than that.

If you're a programmer, feel free to check out the source code too!
